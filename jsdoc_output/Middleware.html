Class: Middleware

Middleware()

Middleware contains functions run before certain express paths. Each function must be called after eg. app.get('profiles', middlewareFunction (req,res) => {})

(async) addAccountIdToRequest(req:, res:, next:)

Function checks request for user authorization, verifies the JWT token, and inputs the user's account id into req.app.locals.accId
